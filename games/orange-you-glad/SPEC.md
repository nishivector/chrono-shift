# Orange You Glad - Game Specification

## Concept
A simple arcade catching game inspired by the Reddit observation that orange balls in ball pits always seem to be the most broken.

## Core Mechanic
- Player controls a basket/catcher at the bottom of the screen
- Tap/click to move the catcher left/right
- Catch orange balls for +10 points
- Avoid other colors (red, green, blue, yellow, purple)

## Visual Style
- Dark blue/navy gradient background
- Colorful ball graphics with shine effects
- Retro arcade feel with Bangers font

## Levels
- Score-based level progression
- Faster ball spawning at higher levels
- Target score increases per level

## Accessibility
- Touch support for mobile
- Mouse/click support for desktop
- Responsive canvas sizing

## Technical
- Single HTML file
- Canvas-based rendering
- Particle effects on catch
- No external dependencies (except Google Fonts)
